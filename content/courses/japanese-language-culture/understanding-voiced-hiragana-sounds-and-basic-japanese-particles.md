# Understanding Voiced Hiragana Sounds and Basic Japanese Particles

Voiced Hiragana sounds are created by adding marks to basic Hiragana characters,
changing sounds such as **ka → ga** or **sa → za**. This lesson also explains how
the particles **no (の)** and **mo (も)** work to express possession and the idea
of "also" in simple Japanese sentences.

## Voiced Hiragana (Dakuten and Handakuten)

Japanese Hiragana has a basic set of sounds. Additional sounds are created by
adding small marks to certain characters.

Two marks are important:

- **Dakuten (゛)** — adds a voiced consonant sound
- **Handakuten (゜)** — creates the **p** sound

### Dakuten Changes

| Base Sound | Voiced Sound |
| ---------- | ------------ |
| か (ka)    | が (ga)      |
| き (ki)    | ぎ (gi)      |
| く (ku)    | ぐ (gu)      |
| け (ke)    | げ (ge)      |
| こ (ko)    | ご (go)      |

The same transformation appears in other rows.

| Base     | Voiced  |
| -------- | ------- |
| さ (sa)  | ざ (za) |
| し (shi) | じ (ji) |
| す (su)  | ず (zu) |
| せ (se)  | ぜ (ze) |
| そ (so)  | ぞ (zo) |

| Base     | Voiced  |
| -------- | ------- |
| た (ta)  | だ (da) |
| ち (chi) | ぢ (ji) |
| つ (tsu) | づ (zu) |
| て (te)  | で (de) |
| と (to)  | ど (do) |

### B and P Sounds

The **ha-series** can transform into **b** or **p** sounds.

| Base    | Dakuten | Handakuten |
| ------- | ------- | ---------- |
| は (ha) | ば (ba) | ぱ (pa)    |
| ひ (hi) | び (bi) | ぴ (pi)    |
| ふ (fu) | ぶ (bu) | ぷ (pu)    |
| へ (he) | べ (be) | ぺ (pe)    |
| ほ (ho) | ぼ (bo) | ぽ (po)    |

## Vocabulary Using Voiced Sounds

Learning vocabulary reinforces the pronunciation patterns.

Examples:

- **がっき (gakki)** — musical instrument
- **ぎんこう (ginkō)** — bank
- **ぐんじん (gunjin)** — soldier
- **げんかん (genkan)** — house entrance
- **ごみ (gomi)** — garbage

Examples with **z-series** sounds:

- **ざんぎょう (zangyō)** — overtime
- **じんこう (jinkō)** — population

Examples with **b-series** sounds:

- **ぶた (buta)** — pig
- **べんとう (bentō)** — lunch box

Examples with **p-series** sounds (often loanwords):

- **パン (pan)** — bread
- **ピンク (pinku)** — pink
- **ポット (potto)** — pot or kettle

## Particle の (No): Expressing Possession

The particle **no (の)** indicates ownership or belonging.

Basic structure:

Noun 1 + の + Noun 2

Meaning: **Noun 2 belongs to Noun 1**

Examples:

- わたしのペンです
  Watashi no pen desu
  "It is my pen."

- たなかさんのくるまです
  Tanaka-san no kuruma desu
  "It is Tanaka's car."

- わたしのともだちのペンです
  "It is my friend's pen."

The particle clearly links nouns to show relationships.

## Particle も (Mo): Meaning "Also"

The particle **mo (も)** means **also** or **too**.

It replaces the topic particle **wa (は)** when expressing similarity.

Example:

Statement 1
わたしは インドじん です。
"I am Indian."

Statement 2
わたしも インドじん です。
"I am also Indian."

Examples:

- たなかさんは がくせいです。
- キムさんも がくせいです。

Meaning:

"Tanaka is a student."
"Kim is also a student."

A previous statement must usually exist before using **mo**.

## Asking About Specialization

A useful academic question pattern is:

せんもん は なん ですか。
**Senmon wa nan desu ka?**

Meaning:

"What is your specialization?"

Examples:

- わたしの せんもん は ぶつりがく です。
  My specialization is physics.

- わたしの せんもん は かがく です。
  My specialization is chemistry.

Students commonly use this expression when introducing their field of study.
