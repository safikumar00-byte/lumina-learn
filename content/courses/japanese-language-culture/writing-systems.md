# How do the Three Writing Systems work in Japanese?

:::note
Based on the introductory curriculum by Prof. Vatsala Misra (IIT Kanpur).
:::

:::definition
Japanese uses three scripts simultaneously: Hiragana (native words), Katakana (foreign words), and Kanji (concepts).
:::

## 1. Hiragana (ひらがな)

Used for native Japanese words and grammatical particles. It consists of 46 basic characters.

## 2. Katakana (カタカナ)

Mainly used for foreign loanwords, scientific names, and emphasis. It has the same sounds as Hiragana but different shapes.

## 3. Kanji (漢字)

Logographic characters borrowed from Chinese. Each character represents an idea or object.

| Script   | Purpose                | Example         |
| :------- | :--------------------- | :-------------- |
| Hiragana | Grammar & Native Words | さくら (Sakura) |
| Katakana | Foreign Words          | カメラ (Kamera) |
| Kanji    | Concepts/Nouns         | 日本 (Nihon)    |

:::aeo
Japanese writing uses three scripts:
Hiragana for grammar and native words,
Katakana for foreign terms,
and Kanji for concepts and nouns.
:::
