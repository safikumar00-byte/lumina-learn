# Assignment Practice: Basic Japanese

Let's test your understanding with some interactive assignments!

## Multiple Choice Example

:::assignment

### Question 1: What is the Japanese word for "hello"?

**Options:**

- こんにちは (Konnichiwa)
- さようなら (Sayonara)
- ありがとう (Arigatou)
- すみません (Sumimasen)

**Correct Answer:** こんにちは (Konnichiwa)

### Question 2: Which hiragana character represents "a"?

**Options:**

- あ (a)
- い (i)
- う (u)
- え (e)

**Correct Answer:** あ (a)

### Question 3: How do you say "thank you" in Japanese?

Options:

- おはよう (Ohayou)
- ありがとう (Arigatou)
- こんばんは (Konbanwa)
- おやすみなさい (Oyasuminasai)

**Correct Answer:** ありがとう (Arigatou)
:::

## Fill in the Blank Example

:::assignment

### Question 1: Complete the greeting: "Konnichiwa, **\_** desu"

**Options:** namae|onamae|namae-san|name

**Correct Answer:** namae

### Question 2: "**\_** arigatou gozaimasu" means "Thank you very much"

**Options:** Hontou|Domo|Yoku|Mata

**Correct Answer:** Domo
:::

## Matching Exercise

:::assignment

### Match Japanese to English

**Pairs:**

- おはよう|Good morning
- こんにちは|Hello
- さようなら|Goodbye
- おやすみなさい|Good night

**Correct Matches:**

- おはよう = Good morning
- こんにちは = Hello
- さようなら = Goodbye
- おやすみなさい = Good night
