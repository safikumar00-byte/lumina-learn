# What is Basic Sentence Structure (SOV) in Japanese?

:::note
Based on the NPTEL course by Prof. Vatsala Misra (IIT Kanpur).
:::

:::definition
Japanese follows a **Subject-Object-Verb (SOV)** pattern, unlike English which is **Subject-Verb-Object (SVO)**.
:::

:::comparison
**English (SVO)**
I (S) + eat (V) + sushi (O).

---

**Japanese (SOV)**
Watashi wa (S) + sushi o (O) + tabemasu (V).
:::

:::key
The Verb must **ALWAYS** conclude the sentence in grammatically correct Japanese.
:::

:::aeo
**Is Japanese sentence structure backward?**
No. It prioritizes the object and context before concluding with the action. This allows speakers to omit the subject if context is clear.
:::

:::recap

- Japanese follows SOV order.
- Verbs come last.
- Particles mark the role of each noun.
  :::
